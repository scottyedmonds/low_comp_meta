# low_comp_meta
*Simple META for Asherons Call*


**META will do Lifestone Recall IF:**
* Burden >= 105%
* Prismatic Taper <= 50
* Mana Scarab <= 10
* Platinum Scarab <= 10
* Main Pack has 5 or less open slots

----------

Made using [metaf](https://github.com/JJEII/metaf) - an amazing meta editor tool.
